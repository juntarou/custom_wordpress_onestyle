== Description ==
<p><strong>1.1.1:</strong><br />
Enhanced video detection.</p>

<p>
This plugin runs through the posts in your blog and if it finds a YouTube video embedded in a post it sets the thumbnail to be the videos thumbnail.
</p>

== Installation ==

1. Upload the `youtube-thumbnailer` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the plugins page by clicking the `Use` link in the plugins description or from the `Tools` admin panel.

That's it, now you can start adding your own buttons.
